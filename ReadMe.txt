===================================================================================
===ReadMe.txt - Instructions for running the program and installing the database===
===================================================================================

1) Running the program
To run the program there are two options
 - double clicking the StockAnalysis.jar file
 - typing the following command into the command prompt/terminal window while in the product directory: "java -jar StockAnalysis.jar" (do not include quotation marks)

2) Installing the database
The database is already installed, as the folder StockInformation, with some data already. But to setup a new/blank database one of the following two options can be taken after deleting the already existing database folder.

 - double clicking the Install.jar file
 - typing the following command into the command prompt/terminal window while in the product directory: "java -jar Install.jar" (do not include quotation marks)
